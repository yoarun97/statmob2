# Terrace

A cinematic, data-rich football stats dashboard. EPL-first, historically deep.

## Stack

Vite + React 18 + TypeScript, Tailwind CSS v4, Framer Motion, React Router v7,
TanStack React Query, Zustand, D3, Recharts, React Three Fiber.

## Dev

```bash
npm install
npm run dev
```

## Env

Copy `.env.example` to `.env.local` and fill in API keys.
